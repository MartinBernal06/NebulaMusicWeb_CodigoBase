CSS MASTER CHEAT SHEET

Guía de referencia rápida: Propiedades, Valores y Mejores Prácticas

Modelo de Caja (Box Model)

width / height
Ancho y alto del elemento (px, %, rem, vw).
padding
Espacio interno. 10px 20px (vertical horizontal).
margin
Espacio externo. 0 auto centra horizontalmente.
border
Borde. 1px solid #ccc.
box-sizing
border-box (CRUCIAL): Incluye padding/border en el ancho total.
min/max-width
Límites de tamaño. 100% evita desbordes.
overflow
Controla contenido sobrante: hidden, scroll, auto.

Flexbox (Layout 1D)

display: flex
Activa el contexto flex.
flex-direction
Eje principal: row, column, row-reverse.
justify-content
Alineación eje principal: center, space-between, flex-end.
align-items
Alineación eje cruzado: center, stretch, baseline.
flex-wrap
Permite saltar línea: wrap, nowrap.
gap
Espacio entre elementos: 1rem.
flex-grow
Factor de crecimiento del hijo. 1 ocupa espacio disp.

Tipografía

font-family
Fuente. 'Roboto', sans-serif.
font-size
Tamaño. Usar rem para accesibilidad.
font-weight
Grosor: 400 (normal), 700 (bold).
line-height
Altura de línea. 1.5 (sin unidad es mejor).
text-align
Alineación: left, center, right, justify.
text-transform
Casing: uppercase, lowercase, capitalize.
letter-spacing
Espacio entre letras (tracking).
text-decoration
none (quita subrayado), underline.
white-space
Control saltos: nowrap evita salto de línea.

CSS Grid (Layout 2D)

display: grid
Activa Grid.
grid-template-columns
Define columnas: 1fr 1fr 1fr o repeat(3, 1fr).
grid-template-rows
Define filas explícitas.
grid-column
Expansión de un hijo: 1 / -1 (todo el ancho).
place-items
Atajo para centrar (align + justify): center.
grid-auto-flow
Dirección flujo auto: dense rellena huecos.

Posicionamiento

position: static
Default. No acepta top/left/z-index.
position: relative
Relativo a sí mismo. Crea contexto para hijos absolutos.
position: absolute
Fuera del flujo. Relativo al padre posicionado más cercano.
position: fixed
Relativo a la ventana (viewport). Se queda al hacer scroll.
position: sticky
Se pega al llegar a un umbral de scroll.
z-index
Orden de apilamiento. Requiere posición no estática.
top / bottom / left / right
Coordenadas de posición.

Fondos y Bordes

background-color
Color sólido (#hex, rgb, hsl).
background-image
url('img.jpg') o linear-gradient(...).
background-size
cover (llena sin deformar), contain (muestra todo).
background-position
Alineación imagen: center center.
border-radius
Redondeo. 50% crea círculo.
box-shadow
Sombra: x y blur spread color.
opacity
Transparencia: 0 a 1. Afecta a hijos.

Efectos y Animación

transition
Suaviza cambios: all 0.3s ease-in-out.
transform
Modifica forma: scale(), rotate(), translate().
animation
Aplica @keyframes: nombre 2s infinite.
filter
Filtros visuales: blur(5px), grayscale(100%).
cursor
Puntero ratón: pointer, not-allowed, grab.
pointer-events
none hace que el click "atraviese" el elemento.

Variables & Trucos

Declarar Variable:

:root { --main-color: #ff0000; }

Usar Variable:

color: var(--main-color);

Recomendaciones Pro:

  • Usa rem para fuentes y márgenes (accesibilidad).
  • Evita !important siempre que puedas.
  • Prefiere Flexbox para 1D y Grid para 2D.
  • Resetea estilos básicos al inicio (box-sizing: border-box).
  • Usa nombres de clase semánticos (BEM o Utility classes).

Pseudo-clases útiles

:hover
Al pasar el ratón por encima.
:focus
Al seleccionar (inputs, botones).
:nth-child(n)
Selecciona el hijo número N (o even/odd).
:not(.clase)
Excluye elementos.
::before / ::after
Pseudo-elementos para insertar contenido cosmético.
